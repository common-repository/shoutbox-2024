A shoutbox plugin for your website!

== Description ==

A customisable shoutbox / chatroom that can be deployed with a simple shortcode

You can chat to users, customise your font colour and send a PM

Custom emojis are supported.
== Installation ==

1. Download the plugin .zip file
2. Login to admin Click Plugins -> Add New -> Upload
3. Find Shoutbox WordPress Plugin in plugin list and activate it.
4. Add `[shoutbox]` anywhere on your site (can only do 1 per page)
5. Now goto Shoutbox under Settings in admin menu and configure it
6. Go to Media > Shoutbox Smilies to manage custom emojis
